# Career & Finance Dashboard

A full-stack web application for tracking your budget and job applications in one place.

## Features

### Budget Tracker
- Track income and expenses with categories
- Visual charts showing spending breakdown
- Real-time balance calculations
- Edit and delete transactions

### Job Application Tracker
- Track job applications with company, position, and status
- Filter by application status (Applied, Interviewing, Offer, Rejected)
- Add notes and dates for each application
- Quick stats overview

## Technologies Used
- React
- Recharts (for data visualization)
- Tailwind CSS
- Persistent browser storage

## How to Run
1. Clone this repository
2. Install dependencies: `npm install`
3. Run the app: `npm start`
4. Open http://localhost:3000 in your browser

## Usage
- Create an account to get started
- Switch between Budget and Job tabs
- All data persists between sessions
